## 포함된 프로젝트 및 예제

* **[calculator-5-3](./calculator-5-3)**: 메모리 기능 구현하기
* **[calculator-5-4](./calculator-5-4)**: 메모리 기능 확장하기
* **[calculator-5-5](./calculator-5-5)**: 복잡한 수식 계산하기

## 실행 방법

각 프로젝트 폴더로 이동하여 실행할 수 있습니다. 예를 들어:

```bash
cd calculator-5-2
cargo run
```
