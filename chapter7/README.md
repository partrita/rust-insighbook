# 7장 - 직접 만든라이브러리 공개하기(도서 관리 프로그램)

이 폴더는 러스트 인사이트북 **7장 - 직접 만든라이브러리 공개하기(도서 관리 프로그램)**의 예제 코드를 포함하고 있습니다.

## 포함된 프로젝트 및 예제

* **[my_library](./my_library)**: 커스텀 라이브러리 크레이트
* **[my_package](./my_package)**: 바이너리와 라이브러리가 포함된 패키지
* **[another_bin](./another_bin)**: 추가 바이너리 바이너리 크레이트 예제
* **[use_remote_crate](./use_remote_crate)**: 외부 크레이트를 의존성으로 가져와 사용하는 예제

## 실행 방법

각 프로젝트 폴더로 이동하여 실행할 수 있습니다. 예를 들어:

```bash
cd my_library
cargo run
```
